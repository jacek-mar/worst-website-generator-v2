## 📖 The Genesis (How This Monstrosity Came to Be) 📖

It started with a simple prompt: *"I like to build 'Worst Website Generator'... with both the app's website itself looking terrible and its functionality—an application that generates equally bad pages."*

And so, the **meta-bad website generator** was born. The irony? We built a *good* app that makes *bad* websites. The cognitive dissonance is intentional. 🤯

### 🎛️ Original Vision (What We Told the AI)

We gave the AI (KiloCode) a detailed specification that included:

| 🎚️ Control | 💥 Effect |
|------------------------|------------------------|
| 🎨 Visual Pain | Color saturation + glow |
| 🧭 Navigation Confusion | Menu logic sabotage |
| 🐌 Performance Nightmare | Artificial delay |
| 🤯 Cognitive Overload | Text density |
| 🤖 AI Absurdity | Copy incoherence |
| 🔥 Total Chaos | Randomization multiplier |

### 💣 Approved Sabotage Library

The project implements these sabotage modules:

1. **🏗️ HTML Structure Sabotage** — Multiple H1s, randomized hierarchy, misuse of divs
2. **🎨 CSS Visual Catastrophe** — Clashing neon colors, aggressive gradients, 4+ fonts
3. **🧭 Navigation Confusion** — Randomized menus, hidden dropdowns, mislabeled links
4. **📝 Form & Auth Frustration** — 15+ fields, absurd captcha, contradictory validation
5. **⚡ JS Behavioral Sabotage** — Delayed clicks, button evasion, reverse scroll
6. **📢 Content Chaos** — Inconsistent tone, over-explanation, random CAPS
7. **⏳ Performance Illusion** — Fake spinners, delayed content
8. **📐 Layout Instability** — Post-load shifts, overlapping sections

## 🎮 What It Does 🎮

The generator is a **5-step wizard** that lets you configure and produce a uniquely terrible website:

### Step 1: 📂 Category
Choose a website type: Portfolio, Startup/SaaS, Product/E-commerce, Finance, Local Business, Event, or Campaign

### Step 2: 📐 Layout
Pick from 5 styles: classic, full-width, split-screen, brutalist, or "coming soon"

### Step 3: 🧱 Sections
Select which page sections to include (Hero and Footer are non-negotiable)

### Step 4: 🎨 Style
Choose a color scheme, font combination, and a chaos level from 0 to 10

### Step 5: ⚙️ Extras
Fine-tune interrupting elements, pick a preset, randomize everything, or set a chaos seed
